Módulo de control de inventario

El módulo de control de inventario ha sido diseñado pensando en las necesidades de las empresas que requieren de un control estricto de las existencias de sus artículos de consumo interno y para la venta.

Control de Inventarios integra todos los movimientos transaccionales que tienen relación con consumos, producción, ventas, ingresos, entre otros, realizados directamente en él o desde los demás módulos con los que interactúa.


Características

  • Gestión de proveedores
  • Creación de catálogos de productos
  • Segmentación de productos
  • Gestión de bodegas
  • Control de existencias de productos
